org-brain

org-brain is mind mapping software based on emacs.

It provides the user with a way of scaffolding data within the terminal.

org-brain projects in this organisation

Infogetics

A project for mapping out software, APIs, etc. for converting between information types.

The focus is on maintaining expressive text as an interchange format.

Sensorium

A project for mapping out software, APIs, etc. for providing some kind of sense to an AI.
