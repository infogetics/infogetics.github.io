org-brain

org-brain is mind mapping software based on emacs.

It provides the user with a way of scaffolding ideas within the terminal.

scaffolding
    Breaking up the learning into chunks and
    providing a tool, or structure, with each
    chunk.

    When scaffolding reading, for example, you
    might preview the text and discuss key
    vocabulary, or chunk the text and then
    read and discuss as you go.
